Installation

Codetrack is created to be used as a portable app. So no installation is required, just copy the files and you're good to go.
However there is one important thing to keep in mind:

Make sure the codetrack files are not placed in a special folder (like 'My Documents', your desktop , a network share,...).
Some processes (like IIS) don't like loading the profiling dll from such a location. I suggest using something like 'c:\temp'
