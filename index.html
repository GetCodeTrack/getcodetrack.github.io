Features

CodeTrack is a versatile profiler with some extra tricks up its sleeve.
On top of the profiling features it also allows debugging and analyzing code execution.

WIDE SUPPORT

  • x64 and x86 architectures
  • .NET 2
  • .NET 4 and up
  • Standalone apps
  • Webservices, WCF, ASP.NET
  • Windows services
  • Including child processes

Analysis

  • Detailed call information including generic arguments
  • Performance tree (per thread/process and combined)
  • Unique timeline view
  • Heli view
  • Code view
  • Search & Mark features

Non invasive & Portable

  • Attach to a running process
  • No installation required, copy the files and you're ready to go!
  • Possibility for offline analysis on a different machine

Different levels of detail

Before profiling you can pick the right detail level, from sampling to in depth tracing (even including arguments).

Possibility to apply filters, only trace what you are interested in.

Log all exceptions that occur during the profiling session.

Free

Best of all: CodeTrack is free to use !

You can use Codetrack for personal and commercial use.
Please refer to the detailed license in the about window.

More to come

Right now the essentials to profile an application are more or less covered, as well as some handy extra features. There's a ton of ideas that are still under development, so stay tuned...

A word from the creator

CodeTrack is a one-man project. It started as a small spike out of curiosity towards the inner working of the .NET framework. As time went by, I started to sense maybe it would be worth to be used by other people. The basic functionalities are covered, and others are currently under development (eg. profiling IIS express, and windows apps). On top of that, there's a long list of cool and unique features still to come!

If you have some issues or have a great idea for a neat feature, feel free to contact me.
This profiler is just a hobby project which means it is being developped only in my spare time. I also have a real daytime job, and I love to spend some time with my wife and kids. Please keep that in mind when I don't immediately get back to you.

CodeTrack is free to use, as well for personal as commercial use. If you like CodeTrack: let me know, that will definately encourage me to go on with this project :-)

I hope that CodeTrack can help you in your daily life as a .NET programmer as much as it does me !

Kind regards,
Nico Van Goethem
